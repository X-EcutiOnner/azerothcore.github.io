Detailed Description

Tests for Unit::resetAttackTimer arithmetic.

Unit::resetAttackTimer resets the swing timer after an attack lands. The timer value determines when the next attack can occur.

Old (buggy) formula: int32 time = int32(GetAttackTime(type) * m_modAttackSpeedPct[type]); m_attackTimer[type] = std::min(m_attackTimer[type] + time, time);

New (fixed) formula: m_attackTimer[type] = int32(GetAttackTime(type) * m_modAttackSpeedPct[type]);

The old formula carried forward negative timer debt, allowing burst attacks after lag spikes or parry-haste timer reductions.
